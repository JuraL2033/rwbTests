## Функционал поиска репозитория через глобальный поиск GitHub

## Версии

Python 3.14.0  
pytest 8.4.1  
playwright 1.55.0  

## Установка

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
playwright install chromium
```

## Запуск
Через терминал:
```bash
pytest tests/test_one.py -v
```
Через PyCharm:
```bash
Запустить tests/test_one.py 
```
