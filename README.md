## Установка

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
playwright install chromium
```

## Запуск
Через терминал:
```bash
pytest tests/test_one.py -v
```
Через PyCharm:
```bash
Запустить tests/test_one.py 
```
